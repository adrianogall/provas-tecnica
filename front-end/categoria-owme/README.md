# Prova Técnica para Front-End

Orientação para desenvolvimento e manutenção da aplicação.

### instalação

```
npm install
```

### Start Server

```
cd /build
```

```
http-server
```

### Start Gulp Watch Dev

```
gulp dev
```

### Build Prod Version

```
gulp build
```

### Tecnologias:

* ES6 via [babel](https://babeljs.io/) (v7)
* SASS [sass-loader](https://github.com/jtangelder/sass-loader)
